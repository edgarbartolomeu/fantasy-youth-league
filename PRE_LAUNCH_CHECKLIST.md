## 🔧 Initial Setup (As Admin)

- [ ] **Open your deployed app**
- [ ] **Login as Admin**
  - [ ] Click "👑 Admin Login"
  - [ ] Set admin password: _______________
  - [ ] Successfully logged in

- [ ] **Upload Player Roster**
  - [ ] Prepared CSV file with all players
  - [ ] Uploaded via Admin → Upload Player Roster
  - [ ] Verified all players appear in Pick Team tab
  - [ ] Number of players: _______

- [ ] **Upload Fixtures**
  - [ ] Prepared CSV with all gameweek fixtures
  - [ ] Uploaded via Admin → Upload Fixtures
  - [ ] Verified fixtures appear in Fixtures tab
  - [ ] Number of gameweeks: _______

- [ ] **Set First Deadline**
  - [ ] Go to Admin → Gameweek Deadlines
  - [ ] Set deadline for Gameweek 1
  - [ ] Date: _____________ Time: _____________
  - [ ] Verified deadline shows in gameweek banner

- [ ] **Export Backup**
  - [ ] Admin → Backup & Restore → Export Data
  - [ ] File saved: `fantasy-league-backup-YYYY-MM-DD.json`
  - [ ] Stored safely (Google Drive, Dropbox, etc.)

## 📅 Weekly Workflow

### Before Each Gameweek
- [ ] Set deadline (at least 1 hour before first match)
- [ ] Remind managers to set teams
- [ ] Export a backup (just in case)

### After Games Are Played
- [ ] Collect match results
- [ ] Upload results via CSV (or enter manually)
- [ ] Verify points calculated correctly
- [ ] Check leaderboard updated
- [ ] Announce results to the group

### Repeat!
- [ ] Set next gameweek deadline
- [ ] Export backup
- [ ] Enjoy the competition!
